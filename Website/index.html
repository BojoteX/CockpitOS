Get Started

Up and Running in Minutes

From download to working panel in five simple steps

Install Arduino IDE

Download Arduino IDE 2.x and install the ESP32 Arduino Core 3.3+ via Boards Manager.

Download CockpitOS

Clone or download from GitHub. Extract to your Arduino folder.

Select a Label Set

Navigate to src/LABELS/ directory and run python generate_data.py

Configure Transport

Edit Config.h to select USB, WiFi, or Serial mode for your setup.

Upload Firmware

Use the Web Flasher to upload directly from your browser, or compile and upload via Arduino IDE.

Configuration

Simple Mapping Files

Define your hardware in clean, readable configuration headers

InputMapping.h 
// Button on GPIO 5
{ "MASTER_CAUTION_RESET", "GPIO", PIN(5), 0, -1,
  "MASTER_CAUTION_RESET", 1, "momentary", 0 },

// 3-position selector switch (group 1)
{ "MASTER_ARM_SW_ARM",  "GPIO", PIN(6),  0, -1,
  "MASTER_ARM_SW", 1, "selector", 1 },
{ "MASTER_ARM_SW_SAFE", "GPIO", PIN(6), -1, -1,
  "MASTER_ARM_SW", 0, "selector", 1 },

// Rotary encoder (CCW/CW pair)
{ "VOLUME_DEC", "GPIO", PIN(8), 0, -1,
  "VOLUME", 0, "variable_step", 0 },
{ "VOLUME_INC", "GPIO", PIN(9), 0, -1,
  "VOLUME", 1, "variable_step", 0 },
LEDMapping.h 
// GPIO LED (active low)
{ "MASTER_CAUTION_LT", DEVICE_GPIO,
  {.gpioInfo = {7}}, false, true },

// WS2812 RGB LED (index 0, dimmable)
{ "LOCK_LT", DEVICE_WS2812,
  {.ws2812Info = {0, 21, 255, 0, 0, 200}},
  true, false },

// PCA9555 expander LED
{ "ECM_FAIL_LT", DEVICE_PCA9555,
  {.pcaInfo = {0x22, 0, 3}},
  false, true },

// TM1637 7-segment display
{ "RPM_DISPLAY", DEVICE_TM1637,
  {.tm1637Info = {9, 8, 0, 0}},
  true, false },
