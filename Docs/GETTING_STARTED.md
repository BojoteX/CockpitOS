## 11. What's Next?

Congratulations! You've successfully set up your development environment and uploaded CockpitOS to your ESP32.

To start building your cockpit panel, you'll need to:

1. **Choose a Label Set** â€” Select an aircraft configuration from the `LABELS/` directory
2. **Configure Your Hardware** â€” Edit `InputMapping.h` and `LEDMapping.h` for your specific buttons, switches, and LEDs
3. **Generate Data Files** â€” Run the Python generator script
4. **Wire Your Panel** â€” Connect physical components to your ESP32's GPIO pins
5. **Run the HID Manager** â€” Use the Python HID Manager on your PC to bridge DCS-BIOS data

See the `README.md` files in the CockpitOS repository for detailed instructions on each of these steps.
