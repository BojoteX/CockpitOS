****** THIS IS FOR AN OLD VERSION OF LovyanGFX ************

There is a BUG in the LovyanGFX library version 1.2.7 when compiling under S3 devices. The file is located here:

Documents\Arduino\libraries\LovyanGFX\src\lgfx\v1\platforms\esp32s3\Bus_RGB.cpp

The Fix for Bus_RGB.cpp is a replacement on line 96-101:

  static void _gpio_pin_sig(uint32_t pin, uint32_t sig)
  {
#if ESP_IDF_VERSION >= ESP_IDF_VERSION_VAL(5, 0, 0)
    gpio_ll_func_sel(&GPIO, pin, PIN_FUNC_GPIO);
#else
    gpio_hal_iomux_func_sel(GPIO_PIN_MUX_REG[pin], PIN_FUNC_GPIO);
#endif
    gpio_set_direction((gpio_num_t)pin, GPIO_MODE_OUTPUT);
    esp_rom_gpio_connect_out_signal(pin, sig, false, false);
  }

And add at the top (after line 26):

#include <esp_idf_version.h>
