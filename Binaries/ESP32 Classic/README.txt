Visit https://cockpitos.bojote.com/upload/?ref=sample and click on "Start" to connect the CockpitOS / ESP32 device and
upload firmware.

Compiled with Serial support. See directory "Serial Manager" and run "connect-serial-port.cmd" there using the COM port your device is listening too after flashing

You can now either enter a mission in DCS -or- simulate a stream/session running "Debug Tools/PLAY_DCS_stream.py"
